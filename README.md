**PythonQuest** é um RPG educacional imersivo projetado para ensinar lógica de programação e a sintaxe da linguagem Python através de uma jornada épica inspirada nos clássicos de 8-bit.

---

## 🎮 O Jogo

Em um mundo onde o **Grande Servidor** foi corrompido, os habitantes de Pythoria perderam a habilidade de codificar. Como um jovem Desenvolvedor Aprendiz, sua missão é explorar reinos devastados, interagir com NPCs mentores e "depurar" o mundo corrigindo erros reais de código Python.

### 🌟 Funcionalidades Principais
- **Motor Python Real:** Integração nativa com **Pyodide (WebAssembly)**, permitindo a execução de código Python real diretamente no navegador.
- **Sistema de Batalha Didático:** Enfrente "Bugs" selvagens que representam erros comuns (TypeError, NameError, SyntaxError).
- **Engine de Mapa Customizada:** Exploração 2D fluida com sistema de colisão e transições de cenário.
- **Progressão RPG:** Ganhe XP e Gold ao resolver desafios, suba de nível e registre suas conquistas na **BugDex**.
- **Interface Responsiva:** Suporte total para teclado e dispositivos móveis via D-Pad virtual.

## 🛠️ Stack Tecnológica

- **Core:** [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Estado Global:** [Zustand](https://github.com/pmndrs/zustand)
- **Backend/Auth:** [Supabase](https://supabase.com/)
- **Motor Python:** [Pyodide](https://pyodide.org/)
- **SFX:** Web Audio API (Sons 8-bit procedurais)

## 🚀 Como Executar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/AndersonFQueiroz/pythonquest.git
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Acesse no navegador:**
   `http://localhost:5173`
